## Current Status Against Requested Standards

This repository is currently a vanilla HTML/CSS/JS app and is not yet fully set up for all requested standards. In particular:

- TypeScript-only workflow is not yet enabled.
- ESLint and Prettier are not yet configured in this repo.
- Automated cSpell checking is not yet wired into scripts/CI.

The control parameters have now been documented in-project and can be used as the baseline for all future changes.

## Working Agreement for Future Changes

- Keep changes small and PR-style.
- Avoid breaking changes.
- Ask before any commit or push.
- Keep docs updated in `docs/CHANGELOG.md` as prompts introduce material changes.
